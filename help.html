APItiser Help

Scan -> Generate -> Download

Quick Start

  • Open a GitHub or GitLab repository page in the active tab.
  • Open APItiser side panel and click Scan Repo.
  • Review detected endpoints, then click Generate Tests.
  • Click Download Tests to save the zip artifact.

Settings You Should Configure

  • Provider/model and API key (OpenAI, Claude, or Gemini).
  • Framework (Jest, Mocha + Chai, or Pytest).
  • Test file directories to skip already covered endpoints.
  • Optional OpenAPI fallback spec for missing static routes.

Context Behavior

APItiser stores context per page. Switching tabs/pages shows that page’s own scan, generated tests, and progress.

Clear Action

Clear removes the current page context: active job, history, generated tests, and artifacts for that page only.

Troubleshooting

  • If scan fails, validate repo access and token scopes in Settings.
  • If generation fails, check provider API key and model selection.
  • Reload the extension from chrome://extensions after updates.
