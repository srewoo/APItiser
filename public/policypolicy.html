APItiser Privacy Policy

Last updated: March 9, 2026

Overview

APItiser runs inside your browser extension context. It scans repository files and generates API tests based on your configuration.

Data Stored Locally

  • Settings (provider/model/framework, repo tokens, test directories).
  • Per-page context state (scan results, generated tests, progress, artifacts metadata).
  • Artifacts for download, stored in extension local storage until cleared.

External Requests

  • GitHub/GitLab API requests for repository scanning.
  • LLM provider API requests (OpenAI/Anthropic/Gemini) for test generation.

API Keys and Tokens

Keys/tokens are stored in chrome.storage.local under the extension context and are not sent to any APItiser backend.

Context Isolation

APItiser isolates state by page context. Clearing one page context does not clear other page contexts.

Retention and Deletion

  • Use Clear in the extension UI to delete current page context data.
  • Removing the extension deletes all extension-local data.
